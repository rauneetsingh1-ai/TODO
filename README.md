# OTP Todo Application

A secure Todo application built using HTML, CSS, JavaScript, Google Apps Script, and Google Sheets. Users authenticate with Email OTP before accessing their personal task list.

## Features

- Email OTP Authentication
- Add, Edit and Delete Tasks
- Mark Tasks as Complete / Undo
- Persistent Login
- Loading States
- Delete Confirmation
- Google Sheets as Database

 ## Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript

**Backend**
- Google Apps Script

**Database**
- Google Sheets

- ## Future Improvements

- Toast Notifications

- ## How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
3. Configure your Google Apps Script Web App URL in `script.js`.
4. Deploy your Apps Script backend.
5. Start using the application.
